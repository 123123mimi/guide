PC·모바일 학습 방법 및 산업안전교육 수강 안내

최종 업데이트: 2026.03
PC 이용가이드 모바일 이용가이드 산업안전교육 가이드
PART 1.

PC 이용가이드

01.

홈페이지 검색

검색창 또는 주소창에 포켓클래스를 검색하세요. (pocketclass.co.kr)

02.

가입 없는 로그인

핸드폰 번호를 입력하여 로그인하세요. (가입 불필요)
회사에서 요청한 과정이 개강되며 개강 문자(카카오톡) 안내를 참고하세요.

로그인 예시
아이디: 01012341234
비밀번호: 01012341234!
03.

나의 강의실 이동

나의 강의실에서 수강 중인 내역을 확인하세요.
홈페이지 우측 상단 나의 강의실을 클릭합니다.

04.

수강 중인 과정 확인

수강 중인 과정을 확인하고 강의실에 입장하여 기간 내 수강하세요.

05.

학습 진행

순차적으로 차시를 눌러 강의를 시청하세요.

06.

화면 구성 (나의 강의실)

수강 중인 과정, 복습 중인 과정, 남은 학습일, 수료증 출력 등을 확인할 수 있습니다.

07.

화면 구성 (과정 상세)

학습하기, 수강기간, 진도율, 교안보기, 과제/평가 등을 확인할 수 있습니다.

08.

화면 구성 (학습 화면)

과정명, 차시 선택, 학습 화면, 학습자료, Q&A 등을 확인할 수 있습니다.

PART 2.

모바일 이용가이드

01.

홈페이지 접속

개강 메시지에 적힌 사이트를 클릭하세요.

02.

가입 없는 로그인

핸드폰 번호를 입력하여 로그인하세요. (가입 불필요)

로그인 예시
아이디: 01012341234
비밀번호: 01012341234! (느낌표 입력)
03.

나의 강의실 이동

왼쪽 상단 또는 오른쪽 상단의 버튼을 눌러 나의 강의실 메뉴로 이동합니다.

04.

강의실 입장

수강 중인 과정을 확인하고 강의실에 입장하세요.
수강 중인 과정이 여러 개일 경우 화면을 아래로 내리면 확인 가능합니다.

05.

학습 진행

차시를 클릭하면 강의가 시작됩니다. 1차시부터 순차적으로 수강하세요.

06.

화면 구성 (나의 강의실)

수강 중인 과정, 복습 중인 과정, 남은 학습일, 진도율, 수료증 등을 확인합니다.

07.

화면 구성 (과정 상세)

학습하기, 남은 학습일, 진도율, 과정 상세, 학습 이력, 교안보기 등을 확인합니다.

08.

화면 구성 (학습 화면)

과정명, 차시 선택, 학습 화면, 학습자료, Q&A 등을 확인합니다.

PART 3.

산업안전교육 가이드

01.

홈페이지 검색

검색창 또는 주소창에 포켓클래스를 검색하세요. (pocketclass.co.kr)

02.

가입 없는 로그인

가입 없이 문자에 안내된 계정으로 로그인하세요. PC/모바일 모두 이용 가능합니다.

03.

나의 강의실 이동

나의 강의실에서 수강 중인 내역을 확인하세요.

04.

수강 중인 과정 확인 및 학습하기

수강기간 내 접속일이 포함되어 있는 경우, 수강 중인 과정에서 강의실에 입장합니다.

05.

본인인증 (카카오톡 인증)

안전보건교육은 공단에서 진행하는 본인인증이 필수입니다.
카카오톡, 휴대폰 인증, 아이핀 중 1가지 방법으로 인증합니다.

필수: 본인이 직접 수강하는 과정으로, 반드시 본인인증을 진행해야 합니다.
06.

본인인증 (휴대폰 인증)

본인 명의 휴대폰 이용 시 문자 인증을 진행합니다.
통신사 선택 → 이름/생년월일/번호 입력 → 인증번호 입력

07.

본인인증 (아이핀)

본인 명의 휴대폰 미사용 시 아이핀 ID와 비밀번호를 입력하여 인증합니다.

08.

학습 진행

1차시부터 순차적으로 학습을 진행하세요.

09.

최종평가

진도율 90% 이상 수강 후 최종평가에 응시합니다.

10.

교안 다운로드

시험이 있는 과정은 교안을 다운로드하여 확인할 수 있습니다.

11.

수료증 출력

수료 조건이 충족되면 수료증 출력 버튼이 생성됩니다. 우측 상단에서 다운로드합니다.

12.

복습 중인 과정

기간이 종료된 과정은 복습 중인 과정에서 수강 가능합니다. 수료 시 수료증 출력도 가능합니다.

13.

수강 종료 과정

수강 및 복습 기간이 종료된 과정에서도 수료증 출력이 가능합니다.
