목차

관리 역할에 맞춰 로그인하여 시스템을 관리합니다.

1. 회원 관리회사/사업장/회원/관리자 2. 과정 관리과정마스터/기수/비환급/환급 3. 콘텐츠 관리콘텐츠 등록/영상/문제 4. 학습현황 관리학습자/수료/평가/설문 5. 게시판 관리공지/FAQ/Q&A/자료실 6. 운영/모니터링메시지/로그/대리수강탐지 7. 시스템 관리역할/권한 설정 8. 홈페이지 관리배너 관리

관리자 메인 화면 (대시보드)

대시보드를 통해 현황 및 활동, 알림 등을 빠르게 확인할 수 있습니다.

01.

회원 관리

01.

회사관리

회사 등록 및 전체 목록 다운로드를 관리합니다.

02.

회사관리 → 신규등록

사업자등록증 및 회신 받은 회사 정보 데이터를 입력하고 등록합니다.

03.

회사관리 → 선택삭제

등록된 회사 중 삭제할 회사를 선택하여 삭제합니다.

04.

회사관리 → Excel 다운로드

등록된 회사 리스트를 다운로드할 때 사용합니다. 목적 작성 후 다운로드합니다.

05.

사업장관리

해당 사업장이 속한 소속회사를 선택 후 등록합니다.
1건 등록 시 신규등록, 다수 등록 시 Excel 버튼을 통해 등록합니다.

06.

회원관리 → 회원목록

등록된 회원 전체 현황 페이지입니다.
1명 등록, 전체 다운로드, 회원 삭제, 비밀번호 초기화가 가능합니다.

07.

회원관리 → 신규등록

* 표시된 부분은 필수로 기입한 후 등록합니다.

08.

회원관리 → Excel 다운로드

등록된 회사별 소속 회원 리스트를 다운로드할 때 사용합니다.

09.

회원관리 → 회원일괄등록 (템플릿)

다수 회원을 등록할 때 사용합니다. 템플릿을 다운로드하여 양식에 맞춰 입력합니다.

10.

회원관리 → 회원일괄등록 (입력방법)

필수 값(엑셀 노란색)을 입력합니다.
엑셀입력시참고사항 클릭 시 코드 생성 → 해당 코드를 복사하여 엑셀 양식에 기입합니다.
과정코드 조회 후 기입 시 해당 과정에 학습자가 자동 등록됩니다.

주의: 엑셀 기입 시 공백(띄어쓰기)이 있는 경우 그대로 반영되므로 반드시 공백 여부를 확인 후 업로드합니다.
11.

회원관리 → 회원일괄등록 (업로드)

엑셀 파일 업로드 → 업로드 목적 기재 → 업로드 및 검증 클릭합니다.

12.

회원관리 → 회원일괄등록 (검증결과)

업로드 히스토리 → 상세 버튼 클릭 → 검증 결과를 확인합니다.
모두 정상이면 등록 실행 버튼을 클릭합니다. 오류 또는 중복 해결 후 등록합니다.

13.

회원관리 → 퇴사일괄등록

다수의 회원을 퇴사 처리할 때 사용합니다.
템플릿 내 이름, 핸드폰 번호 기재 후 엑셀 업로드 및 검증합니다.

14.

회원관리 → 통합등록

신규 회사 + 교육담당자 + 회원등록을 동시에 진행할 때 사용합니다.
필수 값(엑셀 노란색)을 기입합니다.

15.

회원관리 → 통합등록 (상세)

신규 회사 + 교육담당자 + 수강생의 정보를 한번에 등록합니다.
해당하는 코드를 복사 후 템플릿에 붙여넣기합니다.

16.

회원관리 → 사업장일괄변경

회원의 소속 사업장을 일괄 변경할 때 사용합니다.

17.

관리자관리

총괄 관리자, 영업관리자, 교육담당자, 사업장관리자, 교·강사를 등록할 수 있습니다.

18.

관리자관리 → 신규등록

* 표시 되어있는 공란은 모두 기입 필수입니다.

19.

영업지사관리

영업지사 등록 버튼을 통해 영업지사를 등록합니다. (예: OO지사, △△지사 등)

20.

영업지사관리 → 등록

* 표시 되어있는 공란은 모두 기입 필수입니다.

21.

영업본부관리

본부장은 총괄관리자 또는 영업관리자 중 어떤 본부에도 속하지 않은 관리자만 등록 가능합니다.

22.

영업본부관리 → 등록

영업본부 등록 버튼을 통해 영업본부를 등록합니다.

02.

과정 관리

01.

과정카테고리

과정 관리에 용이하도록 과정마스터에서 카테고리를 분류합니다.
분류된 과정은 홈페이지에서 강의탐색으로도 활용됩니다.

02.

과정마스터

콘텐츠 관리에서 만들어진 콘텐츠를 기반으로 학습 설정 및 교육 정보를 등록하는 메뉴입니다.

03.

과정마스터 → 등록방법 (기본정보)

과정구분(환급/비환급) 및 대분류/중분류 선택 → 과정명 입력 → 연결 콘텐츠 선택 → 총 학습시간 입력 → 이미지 등록

연결 콘텐츠: 콘텐츠관리에 등록된 정보를 불러오는 기능입니다. 과정마스터 등록 전 필수로 콘텐츠 정보가 있어야 합니다.
04.

과정마스터 → 등록방법 (학습설정/교육정보)

학습 설정: 과정에 대한 세부 설정을 입력합니다.
교육정보: 연결된 콘텐츠 정보가 자동으로 입력됩니다.

주의: 과정마스터 생성 후 콘텐츠 정보를 수정할 경우 자동 반영되지 않으므로 과정마스터에서 수기로 다시 수정해야 합니다.
05.

과정마스터 → 등록방법 (수료기준/반영률)

수료 기준: 과정의 수료 기준에 따라 입력합니다.
반영률(가중치): 반영률 합계는 100%가 되어야 합니다. (평가/과제 없으면 진도 100%)
훈련비: 필요시 등록합니다.
자동발송 설정: 과정 시작일 기준으로 선택한 내용을 발송합니다.

06.

과정마스터 → 평가설정

등록 후 전체 목록에서 과정명을 눌러 상단 탭 평가설정으로 이동합니다.
진행평가: 중간평가 진행 시 설정
최종평가: 최종평가 진행 시 설정
과제: 과제 진행 시 설정
제약조건: 차시별 학습 진도율 인정 % 입력

07.

기수마스터

동일한 시작일~종료일에 진행하는 과정이 다수일 경우 해당 기간을 만들어서 개강 작업 시 활용합니다.
등록한 기수마스터는 과정관리-비환급에서 선택 가능합니다.

08.

순환마스터

원하는 날짜에 자동으로 개강이 진행되는 메뉴입니다.
목표날짜, 과정 선택, 기수 선택, 교강사 선택 후 업체를 추가하여 저장합니다.

09.

과정관리-비환급

업체별 비환급 과정 개강을 진행하는 메뉴입니다.

10.

과정관리-비환급 → 신규등록

과정상태: 승인 → 과정마스터 연결 → 기수마스터 연결 → 임시생성 → 업체/교강사 선택 → 선택생성

11.

과정관리-환급

업체별 환급 과정이 개강될 수 있도록 설정합니다.

12.

과정관리-환급 → 신규등록

과정상태: 승인 → 과정마스터 연결 → 기수마스터 연결 → 임시생성 → 업체/교강사 선택 → 선택생성

03.

콘텐츠 관리

01.

콘텐츠관리 → 콘텐츠 등록

콘텐츠의 기본 정보와 학습 구조를 설정하는 메뉴입니다.
No.는 LMS에 등록된 콘텐츠 개수, 코드는 FTP서버에 등록된 코드를 따릅니다.

02.

콘텐츠관리 → 신규등록

콘텐츠 코드: FTP 서버에 등록된 콘텐츠 번호 / 입력 후 중복확인 필수
구분: MP4 사용 (포켓에서 사용하는 구분은 MP4 고정)
사전진단: 환급과정 시 필요, 그 외 미사용

03.

콘텐츠 영상 등록 (차시 탭)

저장 후 콘텐츠 목록에서 생성한 과정을 선택 → 상단 탭 차시를 눌러 콘텐츠 목차에 따른 영상을 등록합니다.

04.

콘텐츠 영상 등록 (페이지 추가)

+ 페이지 추가를 눌러 목록을 만들고, 차시명과 페이지 내 주소를 입력합니다.

참고: 비환급은 대체로 통영상, 환급과정은 분할된 영상(인트로-학습목표-학습-아웃트로)을 사용합니다.
05.

콘텐츠 영상 등록 (FTP 서버)

콘텐츠 URL은 등록된 FTP서버 주소에서 영상을 끌고 옵니다.
코드별로 영상이 들어가 있고, 영상 순서에 따라 차시를 구성합니다.

주의: FTP 서버에 등록하는 파일명은 한글 사용 불가, 숫자/영문만 사용합니다.
06.

콘텐츠 문제 등록

엑셀 샘플을 다운로드하여 양식에 맞춰 업로드 시 일괄 등록 가능합니다.
대체로 4지 선다형이며 엑셀의 노란색 칸은 필수입니다.

07.

콘텐츠 교·강사 관리

콘텐츠별 교·강사를 등록하는 메뉴입니다. * 표시된 부분은 필수 기재입니다.

04.

학습현황 관리

01.

학습자 관리

학습자의 과정별 수강 현황을 조회·관리할 수 있습니다.
이름/아이디 검색, 조건별 필터, 상세 학습 이력 확인, Excel 다운로드가 가능합니다.

02.

회사별 현황 관리

회사별 과정 운영 현황을 한눈에 조회할 수 있습니다.
과정별 총원, 평균 진도율, 수료 여부 확인 및 결과보고서 출력이 가능합니다.

03.

평가결과 관리

교육 이수 후 평가 결과를 관리하는 메뉴입니다. 평가 종류, 응시날짜, 채점을 진행할 수 있습니다.

04.

과제결과 관리

과제 제출 결과를 관리하는 메뉴입니다. 과정명에 따른 과제를 관리하고 채점을 진행합니다.

05.

수료 관리

교육기간 종료 후 과정별 수료 여부를 처리합니다.
수료 대상 선택 후 조건확인 → 조건 충족 시 수료처리 → 결과보고서 출력 및 수료증 출력이 가능합니다.

06.

수료관리 → 수료증 출력 (과정수료증)

개별: 이수 인원에 맞춰 PDF 다운로드
리스트: 1개의 PDF 파일 안에 개별 이수증 포함
통합: 학습 전체 인원 통합하여 1장으로 출력

07.

수료관리 → 수료증 출력 (차시수료증)

과정명 내 등록된 차시로 수료증을 출력합니다. 차시를 선택하여 PDF를 다운로드합니다.

08.

설문관리

설문지 등록 및 문항을 추가할 수 있는 메뉴입니다.

05.

게시판 관리

01.

공지사항 관리

LMS 내 공지사항을 등록·조회·관리합니다.
학습자 페이지 > 학습지원센터 > 공지사항에서 노출됩니다.

02.

FAQ 관리

자주 질문하는 내용을 FAQ 형태로 등록·관리합니다.
카테고리, 사용 여부, 검색어 등 조건별 검색이 가능합니다.

03.

Q&A 관리

학습자의 문의 내역을 조회·관리합니다.
문의별 답변 상태, 작성자, 등록일 확인 및 답변 등록이 가능합니다.

04.

자료실 관리

LMS 내 학습자료 및 안내문 등을 등록·관리합니다.
첨부파일을 포함한 자료 다운로드 및 수정·삭제가 가능합니다.

06.

운영/모니터링

01.

엑셀 이력 관리

LMS 내 엑셀 업로드 및 다운로드 이력을 관리합니다.
파일명, 처리 유형, 처리 일시 등 기록을 조회할 수 있습니다.

02.

인증로그 관리

학습자의 인증 이력을 조회·관리합니다.
입과, 학습, 과제 제출, 평가 등 인증 기록을 확인할 수 있습니다.

03.

메시지 발송

학습자에게 카카오 알림톡 또는 문자 메시지를 발송합니다.

04.

메시지 발송 → 템플릿 설정

발송 채널(카카오/SMS/LMS)을 선택 후 미리 설정된 템플릿을 선택합니다.

05.

메시지 발송 → 수신자 선택 및 발송

수신자 목록에서 학습자의 회사, 과정, 반을 선택하고 발송합니다.

06.

발송 이력

LMS에서 발송한 알림 메시지 이력을 조회합니다.
발송일, 채널, 발송 상태 확인 및 상세 내용 확인이 가능합니다.

07.

SMS 템플릿

자주 사용하는 문자 내용을 사전 등록하여 일관된 메시지를 발송합니다.

08.

카카오 템플릿

카카오 알림톡 템플릿을 등록 및 수정하여 관리합니다.

09.

SMS 자동발송 설정

학습자에게 발송되는 SMS 자동발송 규칙을 관리합니다.

10.
