01.

관리자 페이지 로그인

01.

관리자 페이지(LMS) 접속

pocketclass.co.kr/companyMng/login으로 접속합니다.
교육담당자 확인 후 로그인합니다.

로그인 예시
아이디: 사업자등록번호 (예: 1112345678)
비밀번호: 사업자등록번호 + ! (예: 1112345678!)
02.

학습현황 관리

01.

학습자 현황 확인

학습현황관리 > 학습자관리를 클릭합니다.
회사명 선택 → 과정명 선택 → 돋보기 클릭(이름+아이디 공란으로 클릭 시 모든 인원 확인)
학습 진도율, 평가·과제 진행 및 총점 등 학습자 진행 현황을 확인합니다.

03.

수료증 발급

01.

수료증 출력 대상 선택

학습현황관리 > 수료관리를 클릭합니다.
회사명 선택 → 과정명 선택 → 돋보기 클릭 → 출력 인원 체크 → 수료증 출력

02.

수료증 내려받기

수료증 유형: 과정명 또는 차시명으로 출력
수료증 형태:
· 개별: 압축 파일 안에 수강생 개별 수료증 포함
· 리스트: 1개의 PDF 파일 안에 개별 수료증 포함
· 통합: 전체 인원 통합 1장 출력
옵션: 차시 미포함(기본) / 차시 포함 선택 가능

04.

결과보고서 출력

01.

결과보고서 확인

학습현황관리 > 수료관리를 클릭합니다.
회사명 선택 → 과정명 선택 → 돋보기 클릭 (교육기간 종료 및 이수처리 후 과정별 다운로드)
결과보고서 출력을 클릭합니다.

02.

결과보고서 저장

결과보고서 출력 클릭 시 보고서 팝업이 확인됩니다.
PDF 저장을 클릭하여 다운로드합니다.
