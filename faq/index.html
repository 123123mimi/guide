01.

[교육담당자]

Q
Q1. 신규 입사자 교육을 등록하려면 어떻게 해야 하나요?

아래 서류를 대표 메일로 보내주세요.

  • 받는 메일: pocketclass1212@naver.com
  • 첨부 서류: 훈련생 명부(소정 양식) + 사업자등록증 사본

접수 후 교육 등록이 완료되면 해당 직원분들께 개강 안내 문자가 발송됩니다.

Q
Q2. 퇴사자가 발생했습니다. 교육 대상에서 제외해 주세요.

고객센터로 퇴사자 성함과 연락처를 알려주시면 교육 대상에서 제외 처리해 드립니다.

메일(pocketclass1212@naver.com)로 보내주셔도 됩니다.

Q
Q3. 교육 관리자 사이트에 접속하고 싶습니다.

고객센터로 연락 주시면 관리자 사이트 주소와 로그인 정보를 메일로 안내해 드립니다.

관리자 사이트와 학습자 사이트는 별도의 주소이오니 주의해 주세요.

관리자 사이트 로그인 안내

- 아이디: 사업자등록번호

- 비밀번호: 사업자등록번호 뒤 5자리 + !

Q
Q4. 관리자 사이트에서 수료증을 출력할 때 차시별 내역도 함께 출력할 수 있나요?

네, 관리자 사이트에서는 수료증 출력 시 차시를 선택하여 세부 내역 포함 출력이 가능합니다.

Q
Q5. 직원 중 일부만 개강 문자를 못 받았다고 합니다.

등록된 번호가 실제 번호와 다를 수 있습니다. 해당 직원분의 정확한 휴대폰 번호를 확인하신 후 고객센터에 번호 변경 및 문자 재발송을 요청해 주세요.

또한, 070 발신번호로 인해 스팸 처리될 수 있으니 직원분들께 스팸함 확인을 안내해 주시기 바랍니다.

02.

[학습자] 로그인/접속

Q
Q6. 로그인이 안 됩니다.

아래 사항을 순서대로 확인해 주세요.

  1. 올바른 사이트 주소로 접속하셨는지 확인해 주세요. 이전(구) 사이트와 신규 사이트는 주소가 다릅니다. 개강 안내 문자에 포함된 링크로 접속해 주세요.
  2. 아이디는 본인 휴대폰 번호(- 없이), 비밀번호는 휴대폰 번호 뒤에 느낌표(!)를 붙인 형태입니다.
    • 예시: 아이디 01012345678 / 비밀번호 01012345678!
  3. 사이트에서 별도로 회원가입하지 마세요. 회원가입을 하시면 새 계정이 생성되어 기존에 등록된 교육 과정과 연결되지 않습니다.
  4. 번호가 변경되신 경우, 기존에 등록된 번호로는 로그인이 불가합니다. 고객센터로 연락 주시면 변경 처리해 드립니다.
  5. 위 방법으로도 해결이 안 되시면 고객센터로 연락 주세요. 비밀번호 초기화를 도와드립니다.
Q
Q7. 로그인은 됐는데 수강 중인 과정이 보이지 않습니다.

아래 사항을 확인해 주세요.

  • 이전(구) 사이트로 접속하신 경우: 신규 포켓클래스 사이트에서는 구 사이트의 과정이 표시되지 않습니다. 개강 문자에 안내된 신규 사이트 주소로 접속해 주세요.
  • 회원가입을 하신 경우: 기존 계정이 아닌 새로 회원가입한 계정으로 접속하면 등록된 과정이 보이지 않습니다. 안내받은 아이디/비밀번호로 다시 로그인해 주세요.
  • 아직 개강 전인 경우: 교육 개강일 이전에는 과정이 표시되지 않습니다. 개강일에 안내 문자가 발송되오니, 그때 접속해 주세요.
  • 교육이 등록되지 않은 경우: 소속 회사의 교육담당자분께서 수강 등록을 해주셔야 합니다. 교육담당자분께 문의해 주세요.
Q
Q8. 사이트 주소를 모르겠습니다. / 강의는 어떻게 듣나요?
  1. 개강 안내 문자에 포함된 사이트 링크를 클릭하여 접속합니다.
  2. 아이디(휴대폰 번호)와 비밀번호(휴대폰 번호!)로 로그인합니다.
  3. 화면 상단의 나의강의실을 클릭합니다.
  4. 수강 중인 과정 목록에서 학습하기 버튼을 눌러 수강을 시작합니다.

문자를 삭제하셨거나 사이트 주소가 필요하시면 고객센터로 연락 주시면 재발송해 드립니다.

03.

[학습자] 개강 안내 문자

Q
Q9. 개강 안내 문자를 못 받았습니다.
  • 개강 문자는 교육 등록 시 입력된 휴대폰 번호로 발송됩니다. 번호가 변경되셨다면 고객센터에 변경 요청 후 재발송 받으실 수 있습니다.
  • 발신번호가 070으로 되어 있어 스팸함으로 분류될 수 있습니다. 스팸 문자함을 한번 확인해 주세요.
  • 재발송이 필요하시면 고객센터로 연락 주세요.
Q
Q10. 휴대폰 번호가 변경되었습니다. 어떻게 해야 하나요?

고객센터로 연락 주시면 번호, 아이디, 비밀번호를 모두 새 번호로 변경 처리해 드립니다.

변경 완료 후 새 번호로 로그인하시면 됩니다.

04.

[학습자] 수강 진행

Q
Q11. 영상을 끝까지 봤는데 진도율이 100%가 안 됩니다. / 다음 차시로 넘어가지 않습니다.
  • 영상 재생이 끝난 후 진도율 반영까지 약간의 시간이 소요될 수 있습니다. 화면을 그대로 두시면 잠시 후 100%로 업데이트되고, 자동으로 다음 차시로 넘어갑니다.
  • 영상을 끝까지 완전히 재생해야 진도율이 100%로 인정됩니다. 중간에 건너뛰시면 반영되지 않습니다.
  • 오래 기다려도 변경되지 않으면 로그아웃 후 다시 로그인하여 확인해 주세요.
Q
Q12. "중복수강이 탐지되었습니다" 팝업이 뜹니다.

인터넷 브라우저에서 포켓클래스 탭(창)을 여러 개 동시에 열어두셨을 때 나타납니다.

해결 방법: 열려 있는 모든 브라우저 탭/창을 닫고, 브라우저를 완전히 종료한 뒤 다시 접속해 주세요.

Q
Q13. 휴대폰으로 강의를 듣다가 PC로 이어서 들어도 되나요?

네, 가능합니다. 동일한 아이디로 로그인하시면 이어서 수강하실 수 있습니다.

단, 동시에 두 기기에서 접속하시면 중복수강 알림이 나올 수 있으니 한 기기에서 로그아웃 후 다른 기기에서 이용해 주세요.

05.

[학습자] 최종평가(시험)

Q
Q14. 최종평가(시험)는 어떻게 응시하나요?
  1. 로그인 후 나의강의실로 이동합니다.
  2. 해당 과정의 학습하기를 클릭합니다.
  3. 차시 목록 맨 아래쪽에 있는 최종평가 응시 버튼을 클릭하여 응시합니다.
Q
Q15. 최종평가 점수가 미달입니다. 재응시할 수 있나요?

네, 재응시 기간 내 최대 3회까지 재응시가 가능합니다.

재응시 방법: 나의강의실 → 학습하기 → 차시 맨 아래 최종평가 재응시 버튼을 클릭해 주세요.

Q
Q16. 시험을 휴대폰으로 볼 수 있나요?

시험 응시는 PC 사용을 권장드립니다. 휴대폰에서는 화면 크기 및 입력 문제로 응시가 원활하지 않을 수 있습니다.

06.

[학습자] 수료증

Q
Q17. 수료증은 어떻게 출력하나요?
  1. 포켓클래스 사이트에 로그인합니다.
  2. 나의강의실 → 해당 과정의 수료증 출력 버튼을 클릭합니다.
Q
Q18. "수료증 발급이 허가되지 않은 업체입니다"라고 뜹니다.

훈련비가 아직 입금되지 않은 경우 수료증 발급이 제한됩니다.

교육 종료 후 계산서 발행 → 훈련비 입금이 완료되면 수료증 출력이 가능해집니다. 훈련비 관련 사항은 소속 회사의 교육담당자분께 확인해 주세요.

Q
Q19. 수료증에 차시(과목)별 세부 내역이 나오지 않습니다.

학습자 사이트에서는 차시 포함 수료증 출력이 불가합니다.

차시가 포함된 수료증이 필요하시면 소속 회사의 교육담당자분께 요청해 주세요. 교육담당자(관리자) 사이트에서 차시를 선택하여 출력하실 수 있습니다.

Q
Q20. 작년도(이전 연도) 수료증을 확인하고 싶습니다.

이전 연도 교육 기록은 이전(구) LMS 사이트에서 확인 가능합니다.

현재 신규 사이트에서는 조회되지 않으니, 고객센터로 연락 주시면 구 사이트 주소와 로그인 방법을 안내해 드립니다.

Q
Q21. 수료증 PDF 파일이 열리지 않습니다.

PDF 파일을 열 수 있는 프로그램(Adobe Acrobat Reader 등)이 설치되어 있지 않으면 열리지 않을 수 있습니다.

PDF 뷰어를 설치하시거나, 다른 브라우저(크롬 등)에서 다시 시도해 주세요.

07.

[학습자] 교안(교육 자료)

Q
Q22. 교안(교육 자료)은 어디서 다운받나요?

로그인 후 나의강의실 → 해당 과정 → 학습하기 화면에서 교안 다운로드가 가능합니다.

Q
Q23. 교안이 세로로만 보입니다. 가로로 볼 수 없나요?

휴대폰으로 보시는 경우, 휴대폰 상단(또는 설정)의 화면 자동 회전 기능을 켜시면 가로로 볼 수 있습니다.

08.

[학습자] 교육 안내

Q
Q24. 매달 교육을 들어야 하나요?

산업안전보건교육 등 일부 과정은 매월 또는 분기별로 정기 수강이 필요합니다.

수강 대상 여부와 교육 일정은 소속 회사의 교육담당자분께 확인해 주세요.

Q
Q25. 교육 시간은 총 몇 시간인가요?

과정마다 교육 시간이 다릅니다. 나의강의실에서 해당 과정의 차시 수와 시간을 확인하시거나, 교육담당자분께 문의해 주세요.

Q
Q26. 회사에서 교육 들으라고 하는데 저는 신청한 적이 없습니다.

소속 회사의 교육담당자분께서 직원분들의 교육을 일괄 신청하신 것입니다.

별도로 신청하지 않으셔도 정상적인 등록이니, 안내에 따라 수강해 주시면 됩니다.
